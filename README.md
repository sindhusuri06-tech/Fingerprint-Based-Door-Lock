# Fingerprint Based Door Lock

This Arduino project uses a fingerprint sensor to verify users and unlock a door using a servo motor. When a registered fingerprint is detected, the servo rotates to unlock the door and automatically locks again after a few seconds.

## Components
- Arduino Uno
- R307/AS608 Fingerprint Sensor
- SG90 Servo Motor
- 16×2 I2C LCD
